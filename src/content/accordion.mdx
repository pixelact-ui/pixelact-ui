## Accordion

A pixel art styled accordion component built on top of shadcn/ui's Accordion component. Features retro styling with dashed borders and pixel fonts for collapsible content sections.

### Installation

Add the Accordion component to your project using the shadcn CLI:

```bash
npx shadcn@latest add @pixelact-ui/accordion
```

This will automatically install the component and its dependencies to your project.
