## Toast

A pixel art styled toast notification component built on top of <code>sonner</code> Toast. Features retro styling, pixel font, and full support for all underlying toast props. Useful for brief messages and alerts.

### Installation

Add the Toast component to your project using the shadcn CLI:

```bash
npx shadcn add @pixelact-ui/toast
```

This will automatically install the component and its dependencies to your project.
