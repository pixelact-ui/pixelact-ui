## Empty

A pixel art styled empty state component. Features retro styling, pixel font, and customizable variants. Useful for displaying empty states in lists, tables, or pages.

### Installation

Add the Empty component to your project using the shadcn CLI:

```bash
npx shadcn add @pixelact-ui/empty
```

This will automatically install the component and its dependencies to your project.
