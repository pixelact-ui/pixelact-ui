## Drawer

A pixel art styled drawer component built on top of <code>vaul</code> Drawer.

### Installation

Add the Drawer component to your project using the shadcn CLI:

```bash
npx shadcn add @pixelact-ui/drawer
```

This will automatically install the component and its dependencies to your project.
