## Kbd

A pixel art styled keyboard shortcut component.

### Installation

Add the Kbd component to your project using the shadcn CLI:

```bash
npx shadcn add @pixelact-ui/kbd
```

This will automatically install the component and its dependencies to your project.

### Basic
<CodePreview
    language="tsx"
    code={`import { Kbd } from "@/components/ui/pixelact-ui/kbd";
export function KbdExample() {
  return (
    <div className="flex flex-col gap-2">
      <p> Press <Kbd>Space</Kbd> to jump.</p>
      <p> Press <Kbd>Enter</Kbd> to submit.</p>
    </div>
  );
}`}

</div>
</CodePreview>
